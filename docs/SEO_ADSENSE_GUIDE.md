# SEO 및 구글 애드센스 설정 가이드

## 개요
배경화면 웹사이트에 SEO 최적화와 구글 애드센스 광고를 성공적으로 적용했습니다.

## 구현된 기능

### 1. SEO 최적화
- **React Helmet Async**: 동적 메타 태그 관리
- **페이지별 SEO**: 각 페이지마다 최적화된 메타 태그
- **구조화된 데이터**: JSON-LD 스키마 적용
- **소셜 미디어 최적화**: Open Graph, Twitter 카드 지원

### 2. 구글 애드센스
- **반응형 광고**: 다양한 화면 크기에 대응
- **전략적 배치**: 상단, 하단, 사이드바 광고
- **성능 최적화**: 비동기 로딩 및 오류 처리

## 설정 방법

### 1. 환경 변수 설정
`.env` 파일에 애드센스 클라이언트 ID를 추가하세요:

```bash
# 구글 애드센스 설정
VITE_ADSENSE_CLIENT_ID=ca-pub-여러분의-퍼블리셔-ID
```

### 2. 애드센스 광고 슬롯 ID 변경
각 페이지의 광고 컴포넌트에서 `adSlot` 값을 실제 광고 슬롯 ID로 변경하세요:

```tsx
<ResponsiveBannerAd 
  adSlot="실제-광고-슬롯-ID" 
  className="ad-banner-top"
/>
```

### 3. 애드센스 승인 과정
1. **구글 애드센스 계정 생성**
2. **사이트 추가 및 승인 요청**
3. **광고 코드 적용** (이미 완료됨)
4. **승인 대기** (보통 1-14일)

## SEO 최적화 내용

### 기본 메타 태그
- `title`: 페이지별 동적 제목
- `description`: 페이지 내용 요약
- `keywords`: 관련 키워드
- `canonical`: 중복 콘텐츠 방지

### Open Graph (소셜 미디어)
- 페이스북, 카카오톡 등에서 링크 미리보기 최적화
- 배경화면 이미지를 썸네일로 활용

### 구조화된 데이터
- 검색엔진이 사이트 구조를 이해할 수 있도록 JSON-LD 스키마 적용
- 웹사이트 정보 및 검색 기능 명시

## 성능 최적화

### 광고 로딩
- **비동기 로딩**: 페이지 로딩 속도에 영향 최소화
- **오류 처리**: 광고 로딩 실패 시 graceful handling
- **반응형 디자인**: 모바일/데스크톱 최적화

### SEO 성능
- **메타 태그 캐싱**: React Helmet의 효율적인 렌더링
- **이미지 최적화**: 배경화면 썸네일을 OG 이미지로 활용
- **구조화된 마크업**: 시맨틱 HTML 구조

## 추가 개선 사항

### 단기 개선
1. **사이트맵 생성**: `/sitemap.xml` 자동 생성
2. **robots.txt**: 크롤링 가이드라인 설정
3. **구글 애널리틱스**: 트래픽 분석 도구 추가

### 장기 개선
1. **페이지 속도 최적화**: 이미지 lazy loading
2. **AMP 페이지**: 모바일 성능 향상
3. **PWA 기능**: 오프라인 지원 및 앱 설치

## 모니터링

### SEO 성과 측정

- **페이지 속도 인사이트**: 로딩 속도 최적화
- **모바일 친화성 테스트**: 모바일 SEO 확인
